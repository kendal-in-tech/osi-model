The Data Link Layer

The Data Link Layer is the second layer of the OSI Model, and is responsible for data transmissions over a physical medium. Raw data is passed from Layer 1 The Physical Layer to The Data Link Layer where it is organized into Frames. Devices found in Layer 2 include:

  • Network Interface Cards (NICs)
  • Switches
  • Wireless Access Points (WAPs)

The Data Link Layer seperates the data received from Layer 1, and organizes it into frames. Headers and Trailors are added to complete the Frame. Headers a include information for transmission, while trailers include information for accuracy and error checking. This process, known as encapsulation ensures data ready for transmission to the next layer The Network Layer
