The Presentation Layer

The Presentation Layer has two main functions:

  • Encrypt Data,
  • Format data to a form that receiving application understands.

Data formatting and compression is also a task handled in Layer 6. Applications at Layer 7 must be able to interpret data received from Layer 6. Once data is prepared for the particular application, it is moved to the Application Layer.
