The Session Layer

The Session Layer is responsible for secure communications between users on two host. Layer 5 acts as a "translator" between two host so that they can communicate with each other. Protocols used at Layer 3 normally handle session management for applications and may include:

  • RTCP - Real-Time Transport Protocol
  • PAP - Passwork Authentication Protocol

The Session Layer is responsible for authentication / token management. The session is established based upon data recieved from Layer 4, enable a user to connect to an application. The Session Layer manages and handles secure data communications betweem two hosts.
