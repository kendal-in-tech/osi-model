The Transport Layer

The Transport Layer is responsible for the process of transmitting data from one host to another host. Information received from Layer 3 provde a path fo data transmission in Layer 4. Connections between host normalle use one of two protocols:

  • TCP - Transport Control Protocol
  • UDP - User Datagram Protocol

TCP - Transport Layer Protocol is a secure transport protocol. While more reliable and secure, TCP has a slower transmission rate. UDP - User Datagram Protocol is connection-less based and less reliable, however its transfer rate is faster.
