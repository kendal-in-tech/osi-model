The Network Layer

The Network Layer is Layer 3 of the OSI Model, and is responsible for transmitting data over logical networks. Where Layer 1 and Layer 2 deals with physical devices, Layer 3 deals with devices that handle logical networking. These devices may include:

  • Internet Protocol - IPv4 IPv6
  • 3G, 4G LTE, 5G
  • Network Address Translation (NAT)

The Network Layer is responsible for connecting networks to other remote networks. The Network layer is where data can be transmitted to networks that are not local to a particular network. Protocols for network routing, secure data transmission, and network translation will be found in Layer 3.
