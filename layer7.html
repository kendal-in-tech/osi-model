The Application Layer

The Application Layer is where users actually interacts with an application such as:

  • iCloud
  • Outlook
  • GMail

The main function of Layer 7 is to provide services such as Mail/Email and Databases and Directories. Protocols used will allow users to access files in remote locations, such as Cloud Based Storage. Web Browers operate at Layer 7.
