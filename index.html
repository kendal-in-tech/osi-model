OSI Model Quiz

1. Which layer of the OSI Model are IP addresses incorporated?

  • Data Link
  • Network
  • Transport
  • Session

2. Which layer is responsible for authentication management?

  • Physical
  • Session
  • Application
  • Presentation

3. At which layer does a hub operate?

  • Application
  • Network
  • Data Link
  • Physical

4. Which layer provides access to email provider? (I.E. Google, Icloud)

  • Network
  • Transport
  • Data Link
  • Application

5. Which layer has Frames?

  • Physical
  • Application
  • Data Link
  • Session
